# 触摸区域映射说明

## 九宫格布局

屏幕被划分为 3x3 的九个区域：

```
┌─────────┬─────────┬─────────┐
│ TOP_    │ TOP_    │ TOP_    │
│ LEFT    │ CENTER  │ RIGHT   │
├─────────┼─────────┼─────────┤
│ MIDDLE_ │ CENTER  │ MIDDLE_ │
│ LEFT    │         │ RIGHT   │
├─────────┼─────────┼─────────┤
│ BOTTOM_ │ BOTTOM_ │ BOTTOM_ │
│ LEFT    │ CENTER  │ RIGHT   │
└─────────┴─────────┴─────────┘
```

## 从左到右阅读（LEFT_TO_RIGHT）

```
┌─────────┬─────────┬─────────┐
│ 上一页   │ 上一页   │ 下一页   │
│         │         │         │
├─────────┼─────────┼─────────┤
│ 上一页   │ 显示/   │ 下一页   │
│         │ 隐藏    │         │
├─────────┼─────────┼─────────┤
│ 上一页   │ 下一页   │ 下一页   │
│         │         │         │
└─────────┴─────────┴─────────┘
```

**逻辑**：
- 右侧区域（右上、右中、右下）= 下一页
- 左侧区域（左上、左中、左下）= 上一页
- 上中 = 上一页
- 下中 = 下一页
- 中心 = 显示/隐藏工具栏

## 从右到左阅读（RIGHT_TO_LEFT）

```
┌─────────┬─────────┬─────────┐
│ 下一页   │ 上一页   │ 上一页   │
│         │         │         │
├─────────┼─────────┼─────────┤
│ 下一页   │ 显示/   │ 上一页   │
│         │ 隐藏    │         │
├─────────┼─────────┼─────────┤
│ 下一页   │ 下一页   │ 上一页   │
│         │         │         │
└─────────┴─────────┴─────────┘
```

**逻辑**：
- 左侧区域（左上、左中、左下）= 下一页
- 右侧区域（右上、右中、右下）= 上一页
- 上中 = 上一页
- 下中 = 下一页
- 中心 = 显示/隐藏工具栏

## 垂直阅读（VERTICAL）

```
┌─────────┬─────────┬─────────┐
│ 上一页   │ 上一页   │ 上一页   │
│         │         │         │
├─────────┼─────────┼─────────┤
│ (无)    │ 显示/   │ (无)    │
│         │ 隐藏    │         │
├─────────┼─────────┼─────────┤
│ 下一页   │ 下一页   │ 下一页   │
│         │         │         │
└─────────┴─────────┴─────────┘
```

**逻辑**：
- 上方三个区域 = 上一页
- 下方三个区域 = 下一页
- 中间左右区域 = 无功能
- 中心 = 显示/隐藏工具栏

## 设计理念

1. **主要翻页区域**：右侧三个区域（左到右）或左侧三个区域（右到左）为下一页区域
2. **辅助翻页区域**：下中为通用下一页区域
3. **上一页区域**：左侧三个区域（左到右）或右侧三个区域（右到左）为上一页区域
4. **上中区域**：始终为上一页
5. **中心区域**：始终用于显示/隐藏工具栏

## 双页模式

在双页模式下，触摸区域映射保持不变，但翻页时会一次跳转两页。
