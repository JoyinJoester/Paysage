# 翻页动画设置指南

## 设置位置

翻页动画设置已集成到**外观设置**界面中:

1. 打开应用设置
2. 进入"外观设置"
3. 滚动到"翻页动画设置"部分

## 可配置选项

### 1. 翻页动画效果

选择你喜欢的翻页动画模式:

- **滑动** (默认): 简单的水平滑动切换,流畅快速
- **覆盖**: 当前页覆盖在下一页之上,逐渐揭开下一页
- **并排**: 当前页和下一页并排显示,一起移动
- **淡入淡出**: 页面通过透明度变化切换,柔和平滑
- **卷曲**: 模拟真实书页翻转的3D效果,最具沉浸感
- **无动画**: 立即切换,适合追求速度的用户

### 2. 动画速度

调整动画播放的速度:

- **快速 (200ms)**: 快速翻页,适合快速阅读
- **正常 (300ms)**: 默认速度,平衡流畅度和速度
- **慢速 (500ms)**: 慢速翻页,更好地展示动画效果

### 3. 边缘灵敏度

控制滑动手势的响应区域:

- **低 (20%)**: 只在屏幕边缘20%区域响应,避免误触
- **中 (40%)**: 在屏幕边缘40%区域响应,平衡便利性和误触
- **高 (100%)**: 全屏响应滑动手势,最大便利性

### 4. 视觉效果

启用或禁用阴影和深度效果:

- **启用**: 显示阴影、深度等视觉反馈,增强真实感
- **禁用**: 简洁的动画,适合追求简约的用户

### 5. 翻页触觉反馈

控制翻页时的震动反馈:

- **启用**: 翻页时提供震动反馈,增强交互感
- **禁用**: 无震动,适合安静环境

## 使用建议

### 推荐配置

**快速阅读模式**:
- 动画效果: 滑动
- 动画速度: 快速
- 边缘灵敏度: 高
- 视觉效果: 禁用
- 触觉反馈: 禁用

**沉浸式阅读模式**:
- 动画效果: 卷曲
- 动画速度: 正常
- 边缘灵敏度: 中
- 视觉效果: 启用
- 触觉反馈: 启用

**省电模式**:
- 动画效果: 淡入淡出
- 动画速度: 快速
- 边缘灵敏度: 低
- 视觉效果: 禁用
- 触觉反馈: 禁用

### 性能考虑

- **卷曲效果**最消耗性能,在低端设备上可能不够流畅
- **滑动**和**淡入淡出**性能最好,适合所有设备
- 禁用**视觉效果**可以提升性能
- **快速**动画速度可以减少动画时间,提升响应速度

## 实时预览

所有设置更改会立即生效,你可以:

1. 修改设置
2. 返回阅读器
3. 尝试翻页查看效果
4. 如果不满意,返回设置继续调整

## 故障排除

### 动画不流畅

1. 尝试切换到**滑动**或**淡入淡出**模式
2. 禁用**视觉效果**
3. 选择**快速**动画速度
4. 关闭其他后台应用

### 误触翻页

1. 降低**边缘灵敏度**到**低**
2. 确保手指从屏幕边缘开始滑动

### 动画延迟

1. 检查设备存储空间是否充足
2. 清理应用缓存
3. 重启应用

## 技术细节

### 动画实现

- 使用 Jetpack Compose 的 `graphicsLayer` 实现硬件加速
- 所有动画都在 GPU 上渲染,确保流畅度
- 智能预加载下一页,减少加载延迟

### 性能优化

- 自动内存管理,防止内存溢出
- 智能缓存策略,平衡性能和内存使用
- 低端设备自动降级动画效果
